# Style Extractor (Gregory)

Executa a leitura do PDF de referência e gera tokens de design do catálogo.

## Uso

```bash
npm run extract:style
```

Para customizar o arquivo de entrada:

```bash
npm run extract:style -- --input="./Exemplos/Revista Inverno 2026_1776266245778 (1).pdf"
```

## Saídas

- `design/gregory-design-tokens.json`
- `design/gregory-design-tokens.css`
